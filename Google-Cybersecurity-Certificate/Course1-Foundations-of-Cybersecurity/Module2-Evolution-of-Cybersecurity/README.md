# Module 2 - The Evolution of Cybersecurity

## Attack Types

### Password Attack
Attempt to access password-secured devices, systems, networks, or data.

| Type | Description |
|------|------|
| Brute Force | Tries all possible password combinations |
| Rainbow Table | Uses pre-computed hash values to crack passwords |

**Domain**: Communication and Network Security

---

### Social Engineering Attack
Manipulation technique that exploits human error to gain private information or access.

| Type | Description |
|------|------|
| Phishing | Deceptive emails to steal information |
| Smishing | SMS-based deception |
| Vishing | Voice-based deception |
| Spear Phishing | Targeted phishing at specific users |
| Whaling | Targets company executives |
| Social Media Phishing | Uses social media info to attack |
| BEC | Impersonates known source for financial gain |
| Watering Hole | Attacks frequently visited websites |
| USB Baiting | Leaves infected USB for victim to find |
| Physical Social Engineering | Impersonates employee/vendor for physical access |

**Domain**: Security and Risk Management

---

### Physical Attack
Security incident affecting both digital and physical environments.

| Type | Description |
|------|------|
| Malicious USB Cable | Infected cable used to compromise devices |
| Malicious Flash Drive | Infected drive deployed to attack systems |
| Card Cloning and Skimming | Copies card data for unauthorized use |

**Domain**: Asset Security

---

### Adversarial Artificial Intelligence
Manipulates AI and machine learning technology to conduct attacks more efficiently.

**Domain**: Communication and Network Security + Identity and Access Management

---

### Supply-Chain Attack
Targets systems, applications, hardware, and/or software in the supply chain to deploy malware.
- Can affect multiple organizations simultaneously
- Security breach can occur at any point in the supply chain

**Domain**: Security and Risk Management, Security Architecture and Engineering, Security Operations
