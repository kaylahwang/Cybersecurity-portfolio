# Module 2 - The Evolution of Cybersecurity

## Common Attacks and Their Effectiveness

### 1. Phishing
Digital communications used to trick people into revealing sensitive data or deploying malicious software.

| Type | Description |
|------|------|
| Business Email Compromise (BEC) | Impersonates a known source to request information for financial gain |
| Spear Phishing | Targets a specific user or group using a trusted-looking source |
| Whaling | Spear phishing targeting company executives |
| Vishing | Voice communication exploitation to obtain sensitive information |
| Smishing | SMS/text messages used to trick users |

---

### 2. Malware
Software designed to harm devices or networks, primarily for financial gain or intelligence advantage.

| Type | Description |
|------|------|
| Virus | Malicious code that hides in files; requires user to initiate |
| Worm | Self-replicates and spreads across networks without user action |
| Ransomware | Encrypts data and demands payment for restoration |
| Spyware | Secretly gathers and sells personal data without consent |

---

### 3. Social Engineering
Manipulation technique that exploits human error to gain private information or access.

| Type | Description |
|------|------|
| Social Media Phishing | Collects target info from social media then initiates attack |
| Watering Hole Attack | Attacks websites frequently visited by a target group |
| USB Baiting | Leaves infected USB for employee to find and plug in |
| Physical Social Engineering | Impersonates employee/vendor to gain physical access |

### Why Social Engineering Works

| Principle | Description |
|------|------|
| Authority | Impersonates figures of power |
| Intimidation | Uses bullying tactics to pressure victims |
| Consensus | Uses others' trust to appear legitimate |
| Scarcity | Implies limited availability to pressure action |
| Familiarity | Builds fake emotional connection |
| Trust | Develops relationship over time to extract information |
| Urgency | Pressures victims to act quickly without questioning |
