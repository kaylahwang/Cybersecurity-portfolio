# Module 1: Security Domains & Risk Management

## 📝 Module Overview
This module explores how cybersecurity work is organized across specialized areas and how analysts identify, assess, and mitigate security issues. It covers the **CISSP’s eight security domains**, their real-world applications, and structured approaches to navigating organizational threats, risks, and vulnerabilities using frameworks like the **NIST Risk Management Framework (RMF)**.

## 🚀 Key Learning Objectives
* Understand how security domains organize the work of security professionals.
* Apply core principles like **Security Posture** and the **Principle of Least Privilege**.
* Clearly differentiate between **Threats, Risks, and Vulnerabilities**.
* Outline and define the 7 official steps of the **NIST Risk Management Framework (RMF)**.
* Identify core **Risk Management Strategies** and recognize real-world threat actor tactics.
